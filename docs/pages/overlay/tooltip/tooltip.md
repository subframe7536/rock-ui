## Import

```tsx
import { Tooltip } from 'moraine'
```

## Slot Structure

Wrapper trigger with a floating content portal containing text and optional keyboard hints.

```text
trigger
└── content (portal)
    ├── text (optional)
    └── kbds (Kbd, optional)
```
