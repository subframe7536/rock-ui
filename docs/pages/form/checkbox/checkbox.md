## Import

```tsx
import { Checkbox } from 'moraine'
```

## Slot Structure

Checkbox indicator with optional label and description.

```text
root
├── container
│   ├── input
│   └── control
│       └── indicator (optional)
└── wrapper (optional)
    ├── label (optional)
    └── description (optional)
```
